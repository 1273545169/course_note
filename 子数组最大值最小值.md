### 思路解析

`做题依据`：若left-right范围内的max-min<=num，则此范围内所有的子数组也均满足条件

普通解法
```python

class Solution:
    def getNum(self, array, num):
        left, right = 0, 0
        res = 0
        while left < len(array):
            while right < len(array):
                if max(array[left:right + 1]) - min(array[left:right + 1]) > num:
                    break
                right += 1

            res += right - left
            left += 1

        return res

```

窗口解法：

```python

class Solution1:
    def getNum(self, array, num):
        left, right, res = 0, 0, 0
        maxQ, minQ = [], []
        while left < len(array):
            while right < len(array):
                # 最大值队列
                if maxQ and maxQ[-1] <= array[right]:
                    maxQ.pop()
                maxQ.append(array[right])
                # 最小值队列
                if minQ and minQ[-1] >= array[right]:
                    minQ.pop()
                minQ.append(array[right])
                # 不满足条件时，break
                if array[maxQ[0]] - array[minQ[0]] > num:
                    break
                right += 1
            # left需要向右移动，故删掉队列中过期元素
            if maxQ[0] == left:
                maxQ.pop(0)
            if minQ[0] == left:
                minQ.pop(0)
            # [left：right-1] 内的子数组，均满足条件
            res += right - left
            # left向右移动
            left += 1
        return res


```
